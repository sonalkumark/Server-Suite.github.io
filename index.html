# Server-Suite

We configure world with Server-Suite

About Server-Suite

Server-Suite is an "Open Source tool" which can be used for server configurataion at server side.
This tool is build in Python 2.7 .
Server-Suite can help to configure many server with cool and smart GUI provided using PyQt4 framework.
Server-Suite can work on 6 operating systems for more than 5+ server configuration options for each.
It's build for linux based operating systems only.
